# Pathways 🧭

A personalised rights and entitlements navigator for vulnerable people navigating the UK system for the first time.

## The Problem

Immigrants, refugees, and asylum seekers in the UK face an overwhelmingly complex system. They need to know what they're entitled to, what to apply for, and in what order. They often have to do this without English fluency and without anyone to guide them.

## What Pathways Does

Pathways asks a user a short set of questions about their situation (visa status, time in UK, family, employment) and generates a personalised, plain-English action plan — translated into their language — telling them exactly what to do next and how to do it.

## Tech Stack

- **Backend:** Python (FastAPI)
- **Frontend:** React (JavaScript)
- **Database:** Supabase (PostgreSQL)
- **AI:** Google Gemini API (free tier)
- **Translation:** LibreTranslate
- **Deployment:** Render + Vercel
